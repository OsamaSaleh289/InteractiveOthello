TYPE     : BUG 1.103 : watso166 STATUS : Complete
NAME     : Hint Fixes
OWNER    : Katrina Watson
PRIORITY : (1,2,3)
ESTIMATE : 3 hours    ACTUAL : 4 hours 
ISSUE    : 
Random and greedy hints have graphics that appear checked when the menu item 
is clicked, and uncheck when clicked again. These checkboxes match the 
highlight color of the hint. Hints now work for more than just until the 
user clicks on the screen and now reocurr every turn until the user unchecks
the box. Greedy and random hints reuse the strategies from Player's 
MoveStrategies. 1.14 has been redone to use the same highlight method. 
When the game is over, the user can still click the menu items but nothing
will highlight on the board. The checkboxes on the menu items will still 
respond.
